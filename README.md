# IT Ticketing System

A lightweight IT support ticketing system built with the PERN stack (PostgreSQL, Express, React, Node.js).

This application allows users to submit IT support requests.

## Features

- Submit IT support tickets
- Categorize requests (Hardware, Software, Network, Account/Login, Other)
- Store ticket data in PostgreSQL
- RESTful API built with Express
- React-based user interface
- Animated success feedback using Framer Motion
- Responsive design with Tailwind CSS

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS
- Framer Motion

### Backend

- Node.js
- Express
- PostgreSQL (Neon)
- REST API
